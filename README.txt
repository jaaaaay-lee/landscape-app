# 조경 현장 점검 앱 — 인터넷에 올리는 방법 (완전 초보용)

이 폴더를 인터넷에 올리면, 아이패드에서 앱처럼 쓸 수 있어요.
노트북에 아무것도 설치하지 않고, 웹 브라우저 안에서만 다 합니다.
순서대로 따라 하세요. 약 20~30분.

준비물: 윈도우 노트북, 이메일 주소, 이 폴더의 zip 파일.

────────────────────────────────────
1단계. GitHub 가입 (코드 저장소)
────────────────────────────────────
1. 노트북 브라우저(크롬/엣지)에서  https://github.com  접속
2. "Sign up" 클릭 → 이메일·비밀번호 입력해서 무료 가입
3. 이메일로 온 인증번호 입력해서 가입 완료

────────────────────────────────────
2단계. 새 저장소(repository) 만들기
────────────────────────────────────
1. 로그인 상태에서 오른쪽 위 "+" → "New repository" 클릭
2. Repository name 칸에  landscape-app  입력
3. "Public" 선택된 상태 그대로 두기
4. 초록색 "Create repository" 버튼 클릭

────────────────────────────────────
3단계. 파일 올리기
────────────────────────────────────
1. 방금 만든 저장소 화면에서  "uploading an existing file"  라는 파란 글씨 클릭
   (안 보이면 "Add file" → "Upload files")
2. 받은 zip 파일의 압축을 노트북에서 먼저 풀어요.
   (zip 파일 우클릭 → "압축 풀기")
3. 압축 푼 폴더 안의 모든 항목
   (index.html, package.json, vite.config.js, src 폴더, 이 README)
   을 통째로 GitHub 업로드 칸으로 드래그해서 끌어다 놓기
   ※ 폴더 안의 "내용물"을 올리는 거예요. 폴더 자체 말고요.
4. 아래 초록색 "Commit changes" 버튼 클릭
5. 파일들이 목록에 나타나면 성공

────────────────────────────────────
4단계. Vercel 가입 + 연결 (호스팅)
────────────────────────────────────
1. 새 탭에서  https://vercel.com  접속
2. "Sign Up" → "Continue with GitHub" 클릭
   (방금 만든 GitHub 계정으로 바로 연결됨. "Authorize" 눌러 허용)
3. 가입되면 "Add New..." → "Project" 클릭
4. 아까 만든  landscape-app  저장소 옆의 "Import" 클릭
5. 아무것도 건드리지 말고 그냥 "Deploy" 버튼 클릭
6. 1~2분 기다리면 "Congratulations" 축하 화면이 떠요
7. 화면에 보이는 주소(예: landscape-app-xxxx.vercel.app)가
   당신의 앱 주소입니다! 클릭하면 앱이 열려요.

────────────────────────────────────
5단계. 아이패드 홈 화면에 앱으로 추가
────────────────────────────────────
1. 아이패드 사파리에서 4단계의 주소를 엽니다
   (주소를 본인 이메일이나 메모로 보내두면 편해요)
2. 화면 위 공유 버튼(네모에 화살표) 누르기
3. "홈 화면에 추가" 선택 → "추가"
4. 끝! 홈 화면에 앱 아이콘이 생기고,
   누르면 전체화면 앱처럼 열립니다.
   카메라, 위치, 날씨, 음성, 자동저장 전부 작동해요.

────────────────────────────────────
나중에 내용을 고치고 싶을 때
────────────────────────────────────
GitHub 저장소에서 파일을 수정하면, Vercel이 자동으로
앱을 다시 만들어줘요. 주소는 그대로입니다.

────────────────────────────────────
참고 / 주의
────────────────────────────────────
- AI 기능(노트 다듬기·요약·번역·조언)은 별도 설정이 필요해서
  이 기본 배포에서는 작동하지 않을 수 있어요. 나머지 기능
  (사진·치수·견적·보고서·날씨·서명·CSV)은 모두 작동합니다.
  AI까지 켜고 싶으면 그때 다시 도와드릴게요.
- 데이터는 그 아이패드 브라우저 안에 저장돼요.
  사파리 기록을 지우면 데이터도 지워지니, 중요한 보고서는
  "내보내기"로 따로 저장해두세요.

막히는 단계가 있으면, 몇 단계에서 무슨 화면이 나왔는지
알려주세요. 거기서부터 다시 안내할게요.
